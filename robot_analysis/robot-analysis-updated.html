시스템 화면

로봇의 동작 궤적, Cycle별 이상 구간, 주요 센서 데이터를 직관적으로 확인할 수 있습니다.

실시간 데모 — 위 화면은 실제 Robot Analysis System 운영 화면입니다.
3D 궤적 시각화부터 이상 탐지, Cycle 분석까지 모든 과정이 실시간으로 처리됩니다.

시스템 개요

로봇 모니터링 CBM 시스템은 실시간 데이터 수집–분석–시각화를 일원화하여 로봇 상태를 정밀 진단하고, 이상 징후를 사전에 탐지합니다. 생산라인 효율과 안정성을 높이는 디지털 트윈형 예지정비 플랫폼입니다.

  • 로봇 제어기·센서 데이터의 실시간 스트리밍 및 통합 관리
  • Cycle 기반 궤적·토크·속도 이상 탐지 알고리즘
  • 3D 시각화 엔진으로 동작 패턴 및 이상 구간 재현
  • 예지정비·품질관리·라인운영 최적화를 위한 분석 리포트 생성

핵심 기능

현장 로봇 데이터를 기반으로 이상을 조기 탐지하고, Cycle 단위 정밀 분석을 수행합니다.

3D

3D 디지털 트윈 시각화

  • 로봇 모델 렌더링
  • Cycle 경로 및 관절 궤적 실시간 표시
  • 0.25×~8× 속도 제어 및 프레임 이동 지원
CBM

이상 탐지 및 진단

  • Torque·Speed·Position 등 센서 기반 이상 탐지
  • Cycle별 이상 점수 산출 및 임계값 경보
  • AI 기반 패턴 분석과 주요 축 표시
Integration

데이터 연계·저장

  • DB/Modbus/TCP-API 실시간 연동
  • 분석결과 자동 저장 및 이력 조회
  • 주간·월간 리포트 자동 생성
산업 전용 맞춤형 알고리즘 — 각 제조 라인 특성에 맞춰 센서 데이터 가중치를 조정하여 현장별 최적의 분석 성능을 제공합니다.
